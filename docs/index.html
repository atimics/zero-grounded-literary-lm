MODEL NOTE

A real, very small language model.

ZERO.3 is a decoder-only character transformer distilled from the frozen ZERO.1 foundation and ZERO.2 literary teachers while learning Shakespeare, William Blake, Aleister Crowley, the King James Bible, and structured multi-speaker channels. Its 4.85 million learned parameters are quantized to 8-bit rows for this page.

The C engine performs embeddings, RMS normalization, rotary attention, GELU feed-forward layers, and sampling inside WebAssembly. It uses no server and no machine-learning runtime.

It learned two related channel operations: answer a message, and compress old memory plus recent turns into new lossy memory. The page feeds that generated memory back into the next exchange and discards those turns. A small C holographic index—adapted from holostuff's deterministic encode-and-recall contract—can reintroduce one relevant older compressed episode. It still cannot retrieve outside facts or match a contemporary large language model. Oddness is part of its honest scale.
