MODEL NOTE

A real, very small language model.

ZERO is a decoder-only character transformer trained from scratch on three public-domain literary collections. Its 4.85 million learned parameters are quantized to 8-bit rows for this page.

The C engine performs embeddings, RMS normalization, rotary attention, GELU feed-forward layers, and sampling inside WebAssembly. It uses no server and no machine-learning runtime.

It can continue prose and verse, but it does not understand instructions reliably, retrieve facts, or match a contemporary large language model. Oddness is part of its honest scale.
